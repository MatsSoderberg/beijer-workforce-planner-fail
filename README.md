# Beijer Workforce Planner – Fullstack MVP v2 (Deploy-ready)

Den här versionen är förberedd för enkel deploy på Render.

## Deploy på Render

**Build command**
```bash
npm install && npm run build
```

**Start command**
```bash
npm run start
```

Backend serverar även frontendens byggda filer, så appen körs på **en enda URL**.

## Lokalt

```bash
npm install
npm run dev
```
